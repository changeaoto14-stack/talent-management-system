# 自動デプロイ手順（3ステップ）

## ステップ1: GitHubにログイン
開いたブラウザページで、GitHubアカウントにログインしてください：
- Email: changeaoto14@gmail.com
- Password: changeaoto14

## ステップ2: Personal Access Tokenを作成
ログイン後、以下の設定で作成：
1. **Note**: "Talent Market Monitor Deploy" （既に入力済み）
2. **Expiration**: 90 days （または No expiration）
3. **Select scopes**: `repo` と `workflow` にチェック（既に選択済み）
4. 画面下部の緑色の **"Generate token"** ボタンをクリック
5. 表示されたトークン（`ghp_` で始まる長い文字列）を**コピー**

⚠️ トークンは一度しか表示されないので、必ずコピーしてください！

## ステップ3: デプロイスクリプトを実行
ターミナルで以下のコマンドを実行：

```bash
cd /Users/aotokazutaka/talent-management-monitor
./deploy.sh
```

Personal Access Tokenの入力を求められたら、ステップ2でコピーしたトークンを貼り付けて Enter キーを押してください。
