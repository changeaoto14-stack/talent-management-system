# タレントマネジメント業界 変化追跡システム

タレントマネジメントシステム各社のホームページ変更、SEO対策、広告配信内容の変化を自動追跡・可視化する静的HTMLサイトです。

## 🌐 公開URL

**GitHub Pages**: https://&lt;username&gt;.github.io/talent-market-monitor/

（公開後、上記の `<username>` をあなたのGitHubユーザー名に置き換えてください）

## 📊 監視対象企業

- 専門用語ガイド（図解説明）
- **カオナビ** - 人材データプラットフォーム
- **タレントパレット** - AI人事プラットフォーム
- SmartHR - 人事・労務DXプラットフォーム
- HRブレイン - 人事評価・タレントマネジメント
- HRMOS - 人事DXプラットフォーム
- One人事 - ワンストップ人事労務
- ジンジャー - 人事労務プラットフォーム
- オフィスステーション - タレントマネジメント
- COMPANY - 統合タレントマネジメント

## ✨ 主要機能

- **📖 専門用語ガイド**: 「タイトル」「説明文」「メインコンテンツ」の意味を図解で説明
- **📋 詳細変更分析**: 経営層・部長クラス向けの詳細なマーケティング戦略分析レポート
- **🔍 SEO対策分析**: ターゲットキーワード、配置戦略、期待効果の詳細
- **📺 広告戦略分析**: プラットフォーム別の予算・ターゲティング・ROI分析
- **💼 ビジネスインパクト**: 競合状況、推定投資額、ビジネスへの影響分析
- **🎯 展開可能な詳細レポート**: クリックで詳細分析を展開・折りたたみ

## 🌐 GitHub Pagesへのデプロイ方法

### 1. GitHubリポジトリを作成
```bash
# 既にローカルでgit initされています
git add .
git commit -m "Initial commit: タレントマネジメント業界変化追跡システム"
```

### 2. GitHubリポジトリにプッシュ
```bash
# GitHubで新しいリポジトリを作成後（推奨名: talent-market-monitor）
git remote add origin https://github.com/<your-username>/talent-market-monitor.git
git branch -M main
git push -u origin main
```

### 3. GitHub Pagesを有効化
1. GitHubリポジトリページの **Settings** タブを開く
2. 左サイドバーから **Pages** を選択
3. **Source** で `main` ブランチを選択
4. **Folder** で `/ (root)` を選択
5. **Save** をクリック

数分後、以下のURLで公開されます:
```
https://<your-username>.github.io/talent-market-monitor/
```

または直接:
```
https://<your-username>.github.io/talent-market-monitor/index.html
```

**カスタムドメインを使用する場合:**
- Settings > Pages > Custom domain で独自ドメインを設定可能
- 例: `talent-monitor.com` など

## 📱 使用方法

1. サイトを開く
2. プルダウンから企業を選択
   - 「📖 専門用語ガイド」で用語の意味を確認
   - 「カオナビ」「タレントパレット」で詳細な分析レポートを閲覧
3. **変更詳細セクション**で各変更項目をクリックして詳細を展開
4. 「サイトを見る」ボタンで実際の企業サイトにアクセス

## 🎨 特徴

### 詳細分析レポートの内容
- **🎯 変更タイプ**: 戦略的リポジショニング、SEO最適化など
- **📝 具体的な変更内容**: 4-5項目の詳細リスト
- **🔍 SEO戦略詳細**:
  - ターゲットキーワード（月間検索数・競合性付き）
  - キーワード配置戦略
  - 技術的施策（構造化データ、ページ速度、モバイル最適化）
  - 期待される効果
- **📺 広告プラットフォーム別戦略**:
  - Google検索広告、ディスプレイ広告、YouTube広告、LinkedIn広告
  - 予算、ターゲティング、CTR目標、ROI目標
- **💼 ビジネスインパクト**: 競合差別化、リード獲得コスト削減
- **🎯 ターゲット層**: 大企業の経営層・人事部門
- **⚔️ 競合状況分析**: 他社との差別化ポイント
- **💰 推定投資額**: サイトリニューアルや広告費の概算
- **📊 ROI目標**: ROAS 300-400%以上など
- **📈 定量指標**: タイトル文字数、キーワード密度
- **🧪 テスト戦略**: A/Bテストの内容
